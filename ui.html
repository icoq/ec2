UI ITEMS

Buttons

Initialize

You can initialize default button styles by adding the class "btn".

Sizes

You can change a button size by adding the suitable class. F.e. btn-xs, btn-xl.

Background colors

You can change a button background color by adding the suitable class. F.e. btn-aqua, btn-green.

All: btn-orange, -aqua, -bisque, -brown, -black, -blue, -blueviolet, -red, -green, -yellow, -yellowgreen, -gold, -khaki, -indigo, -lime, -lightgrey, -olive, -navy, -purple, -salmon, -thistle, -pink

Colors

You can also change a button color by adding the suitable class. Use btn-cwhite, btn-cblack.

More

You can also a full width button and centred button. Use btn-w100 and btn-centred.

Hovers

You can also change a button hover effect by adding the suitable class.

All: hover-def, -scale, -sweep-to-right, -sweep-to-top, -rect-out -shutter-out-horizontal, -underline-from-left, -radial-out, -underline-from-center

Modal windows

Modal windows

You can add modal windows. Copy this code and paste it over the closing "body" tag

<a href="#modalWindow" class="btn jsModalTrigger hover-def">Modal window trigger</a><div id="modalWindow" class="modal"><div class="modal__overlay jsOverlay"></div><div class="modal__container"><div class="modal__content">Modal window content</div><button class="modal__close jsModalClose">&#10005;</button></div></div>
the id of "div" with class="modal" must be the same as in href of "a" tag which has class="jsModalTrigger" with "#" symbol at the start.

Example: href="#modalWindow"

Alerts

Alerts

You can add alert messages. Copy this code and paste it over the closing "body" tag

<div class="alert alert-success"><span class="alert__text">Added to Cart</span></div><div class="alert alert-def"><span class="alert__text">Added to Cart</span></div><div class="alert alert-fail"><span class="alert__text">Added to Cart</span></div>
To make it working you need to add "alert-trigger" class to a html tag which will call the alert popup you added recently.
You can also add to div with "alert" class "alert-success" or "alert-fail" classes to make it green or red.

Sliders

You can make sliders. Copy this code and paste inside "body" tag.

<div class="slider-3"><div>content 1</div><div>content 2</div><div>content 3</div><div>content 4</div><div>content 5</div></div>
You can also change the amount of showing content by changing the class: slider-1, -2, -3,-4, -5, -6
